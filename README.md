# Interview Manager v2

A full-featured interview scheduling and hiring pipeline management tool with email integration, booking links, onboarding flows, and a guided tour.

## Features

- **Interview pipeline** - Track candidates through hiring stages
- **Email integration** - Send interview invites and follow-ups
- **Booking links** - Self-service scheduling for candidates
- **Onboarding flow** - Guided setup for new users
- **Confetti + Tour** - Polished UX with celebratory moments and product walkthrough

## Tech Stack

- **Framework**: Next.js with App Router
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Deployment

Deployed on Vercel. Push to main for automatic deployment.
